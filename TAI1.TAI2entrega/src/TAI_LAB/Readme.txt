For Linux (Terminal):
- To be able to run the program, it is important to comment the line "package TAI_LAB;" in every .java file.

- In the terminal it is good policy to compile first.

javac Main.java Generator.java Fcm.java Authors.java Similarity.java

- After that just use the following command to run the program 

java Main

For Windows:

- Launch your favorite IDE, open the Maven project and just run it.

Important information:
 
- In the folder "testes" there is 2 sub-folders once for each option, TextGeneration for the first choise and TextSimilarity for the second
